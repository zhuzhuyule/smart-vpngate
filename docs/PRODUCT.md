# Product Requirements

## Vision
Provide a free intelligent VPNGate exit manager with stable automatic exit selection.

## Target Users
- Developers
- Privacy users
- Geo-testing users

## Goals
- Free
- Stable
- Country-aware
- Automatic failover
- Easy deployment

## Core Features
- Country filter
- Locked country mode
- Country priority
- Node pool
- Health monitoring
- Dashboard with Ping/ISP/Public IP

## Non-goals
- Commercial residential proxy
- Multi-user routing in V2

## Milestones
V2: Stable single exit.
V3: Provider abstraction.
V4: Multiple providers and advanced routing.
